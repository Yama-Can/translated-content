{{HTTPSidebar}}
HTTP の **`102 Processing`** レスポンスコードは、サーバーがまだ処理を実行中である場合に表示されます。

サーバーはこのレスポンスを返し、ユーザーにまだ処理中であることを示します。

## 状態

```
102 Processing
```
